Cryptographic Attacks

Gram-Schmidt and LLL

Manticore Provisioner

PietCC

Implemented a Rust interpreter and compiler for the Piet esoteric language using LLVM as a backend, with full support for program codel-size inference, white block elimination, warning about nontermination for certain classes of programs, and emitting optimized IR through running LLVM optimization passes.

Maintaining a list of CTF problems (most of which are probably cryptography) that I solved or helped solve

Rust Ray Tracer

Implemented a parallelized ray tracer in Rust following "Ray Tracing In One Weekend".
